# Installation
Untuk mulai menggunakan NestJS, Anda memiliki dua opsi:
- Menggunakan Nest CLI untuk scaffolding projek baru, atau
- Mencloning suatu starter projek.

Keduanya akan menghasilkan hasil yang sama, yaitu menciptakan direktori projek baru yang sudah dilengkapi dengan file-file intial dan modul-modul pendukung dari NestJS, sehingga membentuk struktur dasar konvensional untuk projek tersebut. Menggunakan Nest CLI direkomendasikan untuk pengguna pertama karena prosesnya lebih mudah dan otomatis.

Berikut adalah cara-cara untuk melakukan kedua opsi tersebut:
## Menggunakan Nest CLI
Untuk scaffolding projek baru dengan Nest CLI, Anda dapat menjalankan perintah-perintah berikut:

```bash
npm i -g @nestjs/cli # Jika belum terinstal
nest new <nama_projek> # Buat direktori projek baru dan populasinya dengan file-file initial dan modul-modul pendukung
cd <nama_projek>
npm run start # Jalankan aplikasi
```

## Alternatives
```bash
$ git clone https://github.com/nestjs/typescript-starter.git project
$ cd project
$ npm install
$ npm run start
```
