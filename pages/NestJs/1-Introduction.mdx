# Pengenalan NestJS

NestJS adalah sebuah framework yang dikembangkan untuk membangun aplikasi server-side yang efisien dan skalabel menggunakan Node.js. Ia didesain untuk mempermudah pengembangan aplikasi dengan menggunakan teknologi modern seperti TypeScript dan kombinasi antar paradigma programing seperti Object-Oriented Programming (OOP), Functional Programming (FP), dan Functional Reactive Programming (FRP)12.

### Fitur Utama NestJS
- Penggunaan TypeScript
NestJS sepenuhnya mendukung TypeScript, tetapi masih memungkinkan para pengembang untuk menggunakan JavaScript murni. Dengan demikian, pengembang dapat manfaatkan fitur tipe statis TypeScript untuk mengurangi kesalahan waktu runtime dan meningkatkan kualitas kod

- Modular Architecture
NestJS menggunakan arsitektur modular yang memungkinkan pengembang untuk memecah aplikasi menjadi unit-unit yang lebih kecil dan terorganisir. Ini membuat kode lebih reusable dan mudah dipelihara

- Dependency Injection System
Sistem injeksi dependensi NestJS mempermudah manajemen dependensi antar module, sehingga memfasilitasi skala dan pemeliharaan aplikasi kompleks

- Integrasi dengan Framework HTTP
NestJS menggunakan framework HTTP populer seperti Express (default) dan Fastify opsionally. Meskipun begitu, ia memberikan tingkat abstraksi di atas framework-framework Node.js standar ini, tetapi juga mengekspos API-nya secara langsung bagi pengembang. Hal ini memberikan kebebasan kepada pengembang untuk menggunakan banyak modul pihak ketiga yang tersedia untuk platform dasarnya.


Dalam rangkaian, NestJS tidak hanya sekedar framework biasa; ia adalah alat lengkap untuk membangun aplikasi server-side yang efisien dan skalabel dengan menggunakan kemampuan abstraksi tingkat tinggi dan integrasi yang luas dengan teknologi terkini. Jadi, singkatnya, NestJS adalah sebuah framework yang dirancang untuk memfasilitasi pengembangan aplikasi server-side yang efisien dan skalabel dengan menggunakan Node.js, serta memberikan kemampuan abstraksi tinggi dan integrasi yang luas dengan teknologi terkini seperti TypeScript dan framework-framework HTTP populer.
