## Set Up the .NET Core Project
```cmd
dotnet new webapi -n YourWebApiProjectName
```

Perintah diatas digunakan 
untuk membuat proyek Web API baru dengan .NET Core.

## Install Required NuGet Packages

Untuk bekerja dengan PostgreSQL dan Entity Framework Core dalam proyek 
.NET Core Web API Anda, Anda perlu menginstal paket-paket NuGet yang diperlukan. 
Buka folder proyek Anda di terminal atau command prompt dan jalankan perintah berikut:

```cmd
dotnet add package Npgsql.EntityFrameworkCore.PostgreSQL
dotnet add package Microsoft.EntityFrameworkCore.Tools
```
